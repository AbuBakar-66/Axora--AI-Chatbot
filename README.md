# 🤖 Axora – AI Chatbot

Axora is a fast, intelligent, and interactive AI chatbot built using **Streamlit** and powered by **Groq’s LLaMA 3.3 large language model**. It delivers real-time streaming responses, context-aware conversations, and a smooth chat experience that closely mimics human interaction.

---

## 🚀 Features

- ⚡ Real-time AI responses using Groq API  
- 💬 Clean and modern chat interface with Streamlit  
- 🧠 Context-aware conversations using chat history memory  
- 🔄 Streaming responses for a natural typing effect  
- 🛠️ Lightweight and easy to run locally  
- 🎯 Beginner-friendly and highly customizable  

---

## 🧠 How It Works

Axora sends user input to Groq’s LLM API and receives intelligent responses in real time. The chatbot maintains conversation history using Streamlit session state, allowing it to respond with context-aware and meaningful replies.

---

## 🛠️ Tech Stack

- Python 🐍  
- Streamlit 🌐  
- Groq API ⚡  
- LLaMA 3.3 Model 🧠
